## Podpora NPM Balíčků

Můžete importovat NPM balíčky pomocí `import`, ale ne `require`.

```js
import $ from "jquery";

$("body").css("background-color", "#FF0");
```

## Podpora Více Souborů

NPM Playground podporuje více souborů, které mohou být propojeny pomocí ECMAScript `import` / `export`. Ale buďte
opatrní &ndash; musíte vyloučit tyto přípony:

- .js
- .jsx
- .ts
- .tsx

Takže pokud existuje soubor s názvem `counter.tsx`, import musí být `"./counter"`:

index.tsx

```tsx
import Counter from "./counter";

/// ....
```

counter.jsx

```jsx
export default function Counter({count}) {
	return <h1>Counter: {count}</h1>;
}
```
