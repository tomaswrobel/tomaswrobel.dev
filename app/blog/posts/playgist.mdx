PlayGist je webový editor kódu, který nahrazuje stávající [NPM Playground](npm-playground.mdx). Umožňuje vám vytvářet a
spouštět projekty přímo v prohlížeči bez nutnosti cloudového úložiště. A pokud chcete svou práci uložit, můžete ji
snadno exportovat do GitHub Gistu, ze kterého lze kód znovu spustit.

<Button to="https://playgist.vercel.app">Spustit PlayGist</Button>

### Co je nového?

-   **Evoluce z "NPM Playground":** PlayGist je podobný svému předchůdci, "NPM Playground". Nicméně, přináší významná
    vylepšení odstraněním omezení cloudového úložiště, které bránily jeho předchůdci. S PlayGist můžete kódovat bez obav
    z omezení úložiště, což vám dává svobodu vytvářet a spouštět vaše Front-End projekty efektivně.
-   **Přeprogramovaný design:** Uživatelské rozhraní se může zdát povědomé, ale není to jen jednoduchý facelift.
    PlayGist prošel kompletním přeprogramováním, přechodem z Lit frameworku na Next.js. Tato transformace zajišťuje
    robustnější a plynulejší zážitek, který vám nabízí výkonné vývojové prostředí přímo ve vašem prohlížeči.
-   **GitHub API Rate Limit:** Je důležité si uvědomit, že PlayGist funguje v rámci limitů GitHub API. Nicméně, máme pro
    vás řešení. Jednoduše se přihlaste ke svému GitHub účtu a můžete obejít tato omezení a využít plný potenciál
    PlayGistu.

### Klíčové vlastnosti

-   **Flexibilní vytváření souborů:** Můžete snadno vytvářet soubory JavaScript (JS), JSX, TypeScript (TS) a TypeScript
    JSX (TSX) přímo v PlayGistu. Ačkoli nemůžete vytvářet složky kvůli naší integraci s GitHub Gisty, můžete importovat
    tyto soubory bez jakýchkoli problémů. Jen nezapomeňte zachovat příponu souboru při importu (např.
    `import "./component-a.tsx"`).
-   **ES moduly a Top-Level Await:** PlayGist podporuje moderní vývojové postupy. Musíte používat ES moduly a my plně
    podporujeme top-level `await`. To zajišťuje, že váš kód zůstane aktuální a připravený pro nejnovější webové
    standardy.
-   **Podpora NPM balíčků: Bare Specifiers:** Využijte výhod NPM balíčků kompatibilních s prohlížeči. PlayGist vám
    umožňuje používat bare specifiers pro tyto balíčky, které budou automaticky vyřešeny z Content Delivery Network.
