### HTML

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web
browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as
JavaScript.

### Hello, World!

```html
<!DOCTYPE html>
<html>
	<head>
		<title>Hello, World!</title>
	</head>
	<body>
		<h1>Hello, World!</h1>
	</body>
</html>
```
